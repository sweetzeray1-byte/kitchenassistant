In Delisio, the public-facing feeds — `getDiscoverRecipes`, `getPopularRecipes`, `getCategoryRecipes` in [src/services/supabaseService.ts] — all filter `.is('user_id', null)`. So only "global" recipes (user_id NULL) appear in Discover, Trending/Popular, and category counts.

Recipes generated by a logged-in user are saved with their `user_id`, so they never appear in public feeds. To publish a recipe to the public feed: `update recipes set user_id = null where id in (...)`. Categories with 0 public recipes are hidden by the frontend ("No categories available").
